# Podman
## 安装
bianbu系统自带软件源中含有该包，可以直接安装

``` bash
sudo apt install -y podman
```

## Hello World
从Docker Hub拉取镜像，并输出内容

``` bash
podman run --rm hello-world
```

## Ubuntu
Docker Hub中有支持riscv64架构的镜像，我们可以输入以下命令来使用

``` bash
podman run --rm -it ubuntu bash
```
